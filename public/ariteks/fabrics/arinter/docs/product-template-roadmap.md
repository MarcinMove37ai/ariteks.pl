## 2. Dane z podstrony produktu

- Nazwa handlowa: **ArInter**
- Opis z title: `Tension Fabric Stuctures for Inside Decoration`
- Linia specyfikacji: `250 g/m², 92% Polyester 8% Elastane, Flexible, Dirt Repellent, Antibacterial, Tension Fabric`

### 2.1. Hero / duże zdjęcie

- Brak wykrytego hero image.

### 2.2. Opis

- ArInter can be used by designers and architects to decorate an environment and
- to give function to the area.
- It is possible to add extra function like high visibility. The fabric has odor cleaning photocatalytic function.
- Flame retardant function is usefull for indoor usage.
- Any aestetic properties can be added by creating flat or curved shapes, globes,
- pyramids, spirals or cubes.
