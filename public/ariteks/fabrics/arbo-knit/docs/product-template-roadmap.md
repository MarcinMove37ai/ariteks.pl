  - Public: `/ariteks/fabrics/arbo-knit/images/arbo-knit-main-pic-m__3e66cb32fa.jpg`

### 2.2. Opis

- World's strongest systhetic fabric which contains Pbo (poly phenylene benzobisoxazole)
- 250 g/m², 58% p-Aramid, 40% Pbo, 2% Carbon. ArBo Knit is designed for firefighters's inner t-shirt.
- ArBo Knit is ultra fire resistant fabric
- for firefighters t-shirts.
- Flame resistant (Flame retardant) fabric with EN 11612 certification.
- Antistatic (EN 1149-5)
- ArBo Knit is the best fabric in all categories as t-shirt turnout gear.
