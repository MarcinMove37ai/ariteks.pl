## 2. Dane z podstrony produktu

- Nazwa handlowa: **ArNeo File**
- Opis z title: `High Visible Mesh Fabric for Safety Jacket`
- Linia specyfikacji: `120 g/m², 100% Polyester, High Visible, Mesh Knit Fabric`

### 2.1. Hero / duże zdjęcie

- Fluorescent Mesh Warp Knit Fabric
  - Public: `/ariteks/fabrics/arneo-file/images/arneo-file-main-pic-m__a53e82a8e2.jpg`

### 2.2. Opis

- ArNeo File is 100% polyester texturised, 120 g/m² warp knitted neon colored mesh fabric for safety jackets. It is the most popular mesh fabric for safety jacket usage.
- ArNeo File is EN ISO 20471 certified.
- Neon dyed fabrics they attract more attention than other colors. They absorb uv light and reflect as a visible light.
- As a result they shine more than other objects.
